What are Data Structures?

Data Structures are ways in which computer scientists store and organize data so as to allow their retrieval and modification to be efficient in memory and time.

Common Structures

  • Arrays
  • Linked Lists
  • Trees
    • Binary Search Tree
    • Red-Black Tree
    • AVL Tree
    • Heap
  • Stacks
  • Queues
  • Hash Table

What are Algorithms?

Algorithms are detailed procedures that complete a task. Computer scientists use them to solve well-defined computational problems.

Common Algorithms for specific topics

  • Graph Traversal
    • Breadth-First Search
    • Depth-First Search
    • Dijkstra's Algorithm
    • A* Search
  • Sorting
    • MergeSort
    • QuickSort
